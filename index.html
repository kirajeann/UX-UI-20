Sierra's Race Against Meningitis Website Redesign

To gain a deeper understanding of user preferences for a non-profit website, we immersed ourselves in the user's perspective. By creating a User Journey Map, we identified key areas where we could enhance the overall user experience.

Based on our user interviews and their feedback on the website, we created cards highlighting key points. After creating all the cards, we categorized them based on similarities. These similarities were then transformed into a navigational map, guiding us in designing the most optimal sitemap.

With our new design, we've developed a comprehensive UI style guide that seamlessly integrates the original color palette with an enhanced layout and additional features.

We believe that redesigning the "Sierra's Race Against Meningitis" website will enable users to focus more effectively on the cause and interact with the site in a smoother, more intuitive manner. Currently, visitors come to the website to educate themselves, register, or donate, but the cluttered design makes these actions challenging. By refining the design aspects, visual hierarchy, and photography, we can greatly enhance user engagement and interaction.
